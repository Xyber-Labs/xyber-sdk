# xyber-sdk

Shared Python utilities for building LLM agents with LangGraph.

## Modules

| Module | Purpose |
|--------|---------|
| [model_registry](src/xyber_sdk/model_registry/README.md) | Initialize LLMs from Google and Together AI |
| [mcp_client](src/xyber_sdk/mcp_client/README.md) | Connect to MCP servers for tool access |
| [guardrails](src/xyber_sdk/guardrails/README.md) | Input/output validation for LangGraph |

## Installation

```bash
uv pip install xyber-sdk
```
